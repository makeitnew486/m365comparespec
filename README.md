# M365 영업 가이드 (FBKR)

후지필름비즈니스이노베이션(FBKR) 영업이 고객 제안 시 활용하는 **Microsoft 365 비교/스펙 정리 도구**입니다.

단일 HTML 파일로 동작하며 별도 설치/빌드 과정 없이 브라우저에서 바로 열어 사용합니다.

## 사용 방법

- 로컬: `index.html` 더블클릭
- 웹: GitHub Pages 활성화 시 `https://makeitnew486.github.io/m365comparespec/`

## 수록 내용

- M365 플랜 비교 (E3 / E5 / Teams 미포함 옵션)
- Copilot 옵션 (Copilot Chat / Microsoft 365 Copilot)
- 주요 앱 목록 (AI, 생산성, 협업, 보안 등)

## 가격 출처

[microsoft.com 공식 가격 페이지](https://www.microsoft.com/ko-kr/microsoft-365/enterprise/microsoft-365-plans-and-pricing) (도구 내 각 카드에 링크 표기)
